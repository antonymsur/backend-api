# backend-api for simple backend api

yarn Commands :

- build
- dev
- start
- test:unit

Tech Stack :

- Express middleware
- OpenAPI/Swagger
- Jest for unit testing
- supertest for api testing
- morgan and winston for logging

Prerequisites:

- Node 16.10.0 and Yarn installed
